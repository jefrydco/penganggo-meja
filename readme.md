# Panganggo Meja

> React app demo for displaying table with filter, sort and search

## Getting Started

```sh
# Install dependencies
yarn

# Run dev server
yarn dev

# Build for production
yarn build

# Spin up server to run production build
yarn preview

# Run test case
yarn test

# Run test case with coverage
yarn coverage
```
